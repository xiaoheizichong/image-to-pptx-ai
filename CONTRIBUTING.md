# Contributing

Thanks for helping improve Image to PPTX AI.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
```

## Checks

```bash
pytest
ruff check .
```

## Pull Requests

Please include:

- A short description of the change.
- Tests for behavior changes.
- Screenshots or generated PPTX examples for rendering changes when helpful.
